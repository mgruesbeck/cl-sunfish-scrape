# cl-sunfish-scrape
Trolling craigslist for a sunfish.

# Instructions

Install node:
```shell
$ brew install node
```

Cd to root dir:
```shell
$ cd scrape-cl
```

Install npm dependencies:
```shell
$ npm install
```

Save your gmail u&k safely as env vars. Check `packadge.json` for var names:
```shell
$ magic stuff
```

Start trolling:
```shell
$ npm run troll
```
