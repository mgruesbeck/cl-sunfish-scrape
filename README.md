# cl-sunfish-scrape
Trolling craigslist for a sunfish.

# Instructions

Install node:
```
$ brew install node
```

Cd to root dir:
```
$ cd scrape-cl
```

Install npm dependencies:
```
$ npm install
```

Start trolling:
```
$ npm run troll
```
