# cl-sunfish-scrape
Trolling craigslist for a sunfish.

# Instructions

Install node:
```
$ brew install node
```

Cd to root dir:
```
$ cd scrape-cl
```

Install npm dependencies:
```
$ npm install
```

Save your gmail u&k safely as env vars. Check `packadge.json` for var names:
```
$ magic stuff
```

Start trolling:
```
$ npm run troll
```
